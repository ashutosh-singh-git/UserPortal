Portal Guide

Below mentioned are the technologies used to build this project:

  • Spring Boot is used as the framework on which this is running.
  • Rest Api's have been developed using spring's web framework
  • All the end-points have been secured using Spring Security, except /login and /register end-points.
  • Hibernate has been used as ORM.
  • MySQL is the database where all the entities reside.
  • Bootstrap is used as essential CSS framework.
  • AngularJS v1.6.1 is used as Javascript framework.
  • Apache Tomcat is the default web server for spring boot.

A few things about functionality:

  • Each User can have multiple profiles
  • Only one Profile can be the Default Profile at a time.
  • Each User will be shown Courses based on their current default profile.
  • We can add multiple Profiles as well as Courses attached with them.
